# TOTVS - Fundamentos de Engenharia de Dados e Machine Learning

Repositório prático para organizar estudos, desafios, anotações e projetos desenvolvidos ao longo do bootcamp **TOTVS - Fundamentos de Engenharia de Dados e Machine Learning**, oferecido pela DIO em parceria com a TOTVS.

## Objetivo

Centralizar a parte prática da trilha, registrando a evolução nos seguintes temas:

- Python
- Git e GitHub
- Ciência de dados
- Banco de dados relacionais e ETL
- Excel com Copilot
- Computação em nuvem
- Machine Learning

## Conteúdos da trilha

Estrutura inicial baseada na página pública do bootcamp da DIO, consultada em **21/04/2026**:

1. Princípios de Python e Versionamento de Código com Git e GitHub
2. Introdução à Ciência de Dados e Python
3. Introdução a Banco de Dados Relacionais e ETL
4. Análise de Dados com Excel e Copilot
5. Noções de Computação em Nuvem
6. Fundamentos e Técnicas de Machine Learning

## Como pretendo usar este repositório

- Registrar exercícios e desafios de cada módulo
- Armazenar códigos, consultas SQL, planilhas e notebooks
- Documentar aprendizados e referências importantes
- Consolidar projetos para portfólio
