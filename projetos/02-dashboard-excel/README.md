# 📊 Dashboard de Variação Cambial USD/BRL com Excel + Copilot

## 📌 Visão Geral
Este projeto apresenta a construção de um **dashboard analítico em Excel** baseado nos dados de câmbio gerados em um pipeline anterior desenvolvido em Python.

O objetivo foi explorar o uso do **Microsoft Copilot no Excel** como apoio na análise de dados, construção de métricas e visualizações, simulando um cenário real de consumo de dados para tomada de decisão.

---

## 🎯 Objetivo
- Utilizar dados estruturados de um pipeline ETL
- Construir um dashboard interativo no Excel
- Aplicar métricas financeiras e indicadores analíticos
- Explorar o uso de IA (Copilot) na análise de dados
- Simular um fluxo de consumo de dados em ambiente corporativo

---

## 🤖 Uso de IA (Microsoft Copilot)

O Copilot foi utilizado como assistente para:

### 🔹 Análise de Dados
- Sugestão de métricas relevantes
- Apoio na interpretação de tendências
- Criação de insights a partir dos dados

### 🔹 Construção do Dashboard
- Sugestão de visualizações (gráficos e KPIs)
- Apoio na organização dos dados
- Auxílio na criação de cálculos e indicadores

### 🔹 Produtividade
- Aceleração na criação do dashboard
- Redução de esforço manual na análise

👉 O Copilot foi utilizado como ferramenta de apoio, mantendo o controle analítico e técnico das decisões.

## 📊 Principais Indicadores

O dashboard apresenta:

- 📌 Última taxa de câmbio

- 📅 Data da última cotação

- Máxima
