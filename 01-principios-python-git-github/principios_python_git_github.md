## 🔄 Git e GitHub (Básico)

Git é um sistema de controle de versão e o GitHub é uma plataforma para hospedagem de repositórios.

### Conceitos importantes:
- **Repositório**: onde o código é armazenado
- **Commit**: registro de alterações
- **Branch**: linha de desenvolvimento
- **Merge**: junção de alterações

### Fluxo básico:
```bash
git init
git add .
git commit -m "primeiro commit"
git branch -M main
git remote add origin <url-do-repo>
git push -u origin main
```

### Operações comuns:
- `git status` → verificar alterações
- `git pull` → atualizar repositório local
- `git push` → enviar alterações

### Boas práticas:
- Commits pequenos e descritivos
- Uso de branches para novas funcionalidades
- Versionamento contínuo
