# ☁️ Noções de Computação em Nuvem

## 📌 Visão Geral
Este módulo apresenta os conceitos fundamentais de computação em nuvem, com foco na AWS (Amazon Web Services). São abordados princípios de infraestrutura, redes e armazenamento, essenciais para construção de soluções escaláveis.

---

## 🌐 Como a Computação em Nuvem Funciona

A computação em nuvem permite acessar recursos de TI sob demanda pela internet.

### Principais características:
- Escalabilidade sob demanda
- Pagamento conforme uso (pay-as-you-go)
- Alta disponibilidade
- Acesso remoto

### Modelos de serviço:
- **IaaS** (Infraestrutura como Serviço)
- **PaaS** (Plataforma como Serviço)
- **SaaS** (Software como Serviço)

---

## 🏗️ Entendendo a Infraestrutura AWS

A AWS é uma das principais plataformas de nuvem do mercado.

### Componentes principais:
- **Regiões**: localizações geográficas
- **Zonas de disponibilidade (AZs)**: datacenters isolados dentro de uma região
- **Serviços principais**:
  - EC2 (máquinas virtuais)
  - Lambda (serverless)
  - S3 (armazenamento)

### Benefícios:
- Escalabilidade global
- Alta confiabilidade
- Segurança integrada

---

## 🔗 Redes em AWS

A AWS oferece controle completo sobre redes.

### Principais conceitos:
- **VPC (Virtual Private Cloud)**: rede isolada
- **Subnets**: divisão da rede (pública/privada)
- **Security Groups**: regras de acesso (firewall)
- **Internet Gateway**: acesso à internet

### Exemplo:
- Aplicação web em subnet pública
- Banco de dados em subnet privada

---

## 💾 Armazenamento e Banco de Dados AWS

A AWS oferece diversas soluções para armazenamento e dados.

### Armazenamento:
- **S3**: armazenamento de objetos
- **EBS**: disco para instâncias EC2
- **Glacier**: armazenamento de arquivos arquivados

### Banco de Dados:
- **RDS**: bancos relacionais gerenciados
- **DynamoDB**: NoSQL gerenciado
- **Redshift**: data warehouse

### Vantagens:
- Alta disponibilidade
- Backup automático
- Escalabilidade
